## 2. データ項目一覧（テーブル設計）

### PatientSummary テーブル

| カラム名 | 型 | 必須 | 説明 |
|---------|-----|:----:|------|
| id | UUID | ○ | 主キー |
| patientId | UUID | ○ | 患者ID（外部キー、ユニーク） |
| cautionMedicationRefusal | Boolean | ○ | 服薬拒否・飲み忘れが多い |
| cautionUnderstandingDifficulty | Boolean | ○ | 説明理解が困難 |
| cautionFamilyPresenceRequired | Boolean | ○ | 家族・施設職員の同席が必要 |
| cautionTimeRestriction | Boolean | ○ | 時間帯・曜日の制約あり |
| cautionTroubleRisk | Boolean | ○ | 急な変更・トラブルが起きやすい |
| cautionOther | Boolean | ○ | その他注意あり |
| cautionOtherText | String(100) | - | その他注意の内容 |
| prohibitedActions | String(300) | - | やってはいけないこと（3行まで） |
| approachType | Enum | ○ | 対応方針（normal/careful/contact_first） |
| approachNote | String(100) | - | 対応方針の補足 |
| primaryContactName | String(50) | ○ | 最優先連絡先 氏名 |
| primaryContactRelation | String(30) | ○ | 最優先連絡先 関係 |
| primaryContactPhone | String(20) | ○ | 最優先連絡先 電話番号 |
| secondaryContactName | String(50) | - | 次点連絡先 氏名 |
| secondaryContactRelation | String(30) | - | 次点連絡先 関係 |
| secondaryContactPhone | String(20) | - | 次点連絡先 電話番号 |
| recentChanges | String(300) | ○ | 最近の変化・直近トピック |
| recentChangesUpdatedAt | DateTime | ○ | 直近トピック更新日時 |
| recentChangesUpdatedBy | UUID | ○ | 直近トピック更新者 |
| freeNote | String(500) | - | 自由補足 |
| createdAt | DateTime | ○ | 作成日時 |
| createdBy | UUID | ○ | 作成者 |
| updatedAt | DateTime | ○ | 更新日時 |
| updatedBy | UUID | ○ | 更新者 |

### PatientSummaryHistory テーブル（履歴）

| カラム名 | 型 | 必須 | 説明 |
|---------|-----|:----:|------|
| id | UUID | ○ | 主キー |
| summaryId | UUID | ○ | サマリーID（外部キー） |
| snapshot | JSON | ○ | 変更時点の全データスナップショット |
| changedFields | JSON | ○ | 変更されたフィールド一覧 |
| changedAt | DateTime | ○ | 変更日時 |
| changedBy | UUID | ○ | 変更者 |

## 3. バリデーションルール

### 必須チェック
| フィールド | ルール |
|-----------|--------|
| 注意点チェック（①） | 少なくとも一度保存が必要（全てfalseでも可） |
| その他注意テキスト | cautionOther=true の場合は必須 |
| 対応方針（③） | 選択必須 |
| 最優先連絡先（④） | 氏名・関係・電話番号すべて必須 |
| 最近の変化（⑤） | 1文字以上必須 |

### 文字数制限
| フィールド | 最大文字数 |
|-----------|----------|
| cautionOtherText | 100文字 |
| prohibitedActions | 300文字（3行相当） |
| approachNote | 100文字 |
| primaryContactName | 50文字 |
| primaryContactRelation | 30文字 |
| primaryContactPhone | 20文字 |
| recentChanges | 300文字（3行相当） |
| freeNote | 500文字 |

### 電話番号フォーマット
- 数字、ハイフンのみ許可
- パターン: `/^[0-9\-]+$/`
